Analysis of Titanic dataset:¶

I will be analyzing Titanic dataset to find which variables could have affected survival of passengers in the ill fated ship.

Questions on dataset:

Trying to analyze what factors might have affected the survivability of passengers of Titanic. Athough per common sense, the factors that could affect are: 1.Passenger class 2. Sex 3. Age or/and 4. Embarked. However, I will analyse other parameters that are provided in dataset like SibSp, Parch, ticket, fare or Cabin as well to see what data says...

To Analyze, these are the steps I will follow:

  1. Read dataset as dataframe
  2. Discribe data to get familiar with it
  3. Check which parameters have correlation with survival rate
  4. Analyze every data that could be used by using panda, numpy and visualization module matplotlib of python
  5. Finally, will report the analysis, it's limitations, and probable improvements.

# Check if there is correlation between survival and the passenger's class of travel

def correlation(x,y):
    std_x=(x-x.mean())/x.std(ddof=0)
    std_y=(y-y.mean())/y.std(ddof=0)
  
    return (std_x*std_y).mean()
